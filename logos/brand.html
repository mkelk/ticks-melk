Voice

How we speak. No fluff, no hype.

Direct

Get to the point. Developers don't have time for marketing speak.

Technical

Speak to builders. Use precise terms. Show, don't tell.

Confident

State facts. Avoid hedging. We know what we built.

Lowercase

Brand name is "ticks", not "Ticks". Casual, approachable.

Usage

Keep it consistent. Keep it simple.

Do

  • Use logo on dark backgrounds only
  • Let the logo breathe with clear space
  • Use green as the primary accent
  • Use mono font for code and CLI
  • Write "ticks" in lowercase

Don't

  • Place logo on light or busy backgrounds
  • Stretch, rotate, or modify the logo
  • Overuse accent colors (red, yellow, blue)
  • Use mono font for body text
  • Write "Ticks" or "TICKS"
